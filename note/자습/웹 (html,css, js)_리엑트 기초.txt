html 하이퍼 텍스트 마크업 랭귀지
    빼대를 구성되는 마크업 언어
	마크업 : 문서나 정보처리를 위해 추가되는 정보
	사용용도 : 웹, 웹브라우저 앱
 
    태그 : <>, </>
	해드 : 속성(메타데이터), 타이틀
	바디 : 컨텐츠 

spa : 하나의 페이지만 존재하는 웹 ,어플리케이션
	
css : 웹사이트의 디자인 (15강)

자바스크립트 : ECMAScript
    자료형 : 자료가 대입되는 시점 결정됨(동적타이핑)
	let, ver

	정수형 문자열 블린 널 언디파인 어레이 오브젝트 

    연산자 : 연산을 위해 사용되는 기호
	대입연산자 : =
	사칙연산자 : +, -, *, /, %, **
		+=, -=, *=, /=	
	증가 연산자, 감소연산자 : ++, --
		postfix : 	a++, 증감전 값을 반환 후 변수값 증감
		perfix : 	++a, 변수값 증감 후 증감된 값 반환
	관계연산자(비교연산자) : <, >, <=, >= 왼쪽 기준
		동등연산자 : ==, != 	값 비교
		일치연산자 : ===, !=== 	자료형 까지 비교
		이진 논리연산자 &&(and), ||(or)  
	조건부연산자(삼항 연산자) : 비교할 값 ? true 인경우 : false 인 경우

    함수 : 파라미터(인자) 를 받아 값을 출력함
	function statement	: function 함수명(인자){
				내용;
			  }
	arrow function 	: const 함수명 = (인자)=>{
				내용;
			  }
